TIT Test Portal

TIT Test Portal is a web-based online examination system designed to conduct, manage, and evaluate tests digitally for educational institutions.

Features

Secure login for students and administrators

Create, manage, and schedule online tests

Question and test management

Automated evaluation and result generation

Simple and user-friendly interface

Purpose

The project aims to replace manual examination processes with a reliable digital system, improving efficiency, accuracy, and transparency.

Tech Stack

Frontend: HTML, CSS, JavaScript

Backend: Java / Node.js (update as per your project)

Database: MySQL / MongoDB

How to Run

Clone the repository

Open the project in VS Code

Configure database settings

Run the server

Access the application in the browser
